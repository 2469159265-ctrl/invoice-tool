# 多页小票提取工具

基于 GLM-4.6V 本地模型的多页小票/发票信息提取工具，支持 PDF 和图片批量处理。

## 功能特点

- 支持 PDF 多页提取（每页转为图片）
- 支持多种图片格式（PNG, JPG, JPEG, BMP, TIFF）
- 批量处理多个文件
- 自动提取：购方、销方、发票说明、总金额、币种、日期、发票号码
- 提取不到则为空，不虚构信息
- 输出 Excel 格式，带总金额统计

## 使用方法

1. 双击运行 `多页小票提取工具.exe`
2. 配置 GLM-4.6V 模型 API 地址和密钥
3. 添加 PDF 文件或图片文件
4. 设置输出路径
5. 点击"开始提取"

## 输出字段

| 字段 | 说明 |
|------|------|
| 序号 | 流水号 |
| 来源文件 | 原始文件名 |
| 购方 | 购买方 |
| 销方 | 销售方 |
| 发票说明 | 小票类型描述 |
| 总金额 | 金额数字 |
| 币种 | 如 USD, CNY, EUR |
| 日期 | YYYY-MM-DD 格式 |
| 发票号码 | 收据编号 |

## 系统要求

- Windows 10/11
- 无需 Python 环境（已打包为 exe）

## 技术栈

- PyInstaller 打包
- Tkinter GUI
- PyMuPDF (PDF处理)
- openpyxl (Excel输出)
- GLM-4.6V (OCR/信息提取)
